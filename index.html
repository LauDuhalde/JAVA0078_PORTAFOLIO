Proyecto de base de datos

Este proyecto consiste en el diseño e implementación de una base de datos relacional para gestionar los pedidos, productos y clientes de una tienda de tecnología. A través de scripts SQL, se demuestra cómo registrar clientes, productos disponibles, pedidos realizados y los detalles de cada venta. Además, se incluyen consultas que permiten generar reportes de ventas, controlar el stock, identificar productos no vendidos y mantener actualizada la información mediante operaciones de inserción, actualización y eliminación.

Proyecto de Agenda Digital

Aplicación de consola que permite gestionar tareas: agregar con fecha y descripción, listar todas, filtrar por estado y marcarlas como completadas. El proyecto aplica conceptos de Programación Orientada a Objetos y cuenta con pruebas unitarias en JUnit para garantizar el correcto funcionamiento de sus clases principales.
